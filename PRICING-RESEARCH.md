## Competitor Analysis

### Zep (Direct Competitor - AI Memory Platform)
| Plan | Price | Included | Notes |
|------|-------|----------|-------|
| Free | $0 | 1,000 credits/mo | Low rate limits, varies with load |
| Flex | $25/mo | 20,000 credits | Auto-topup at 20% (+20k), 600 RPM |
| Flex Plus | $475/mo | 300,000 credits | 1,000 RPM, webhooks, 7-day API logs |
| Enterprise | Custom | Custom limits | SOC 2, HIPAA, SLA guarantees |

**Pricing Model:** Credit-based (charged per "episode" of ingestion/processing)
- Unused credits roll over 60 days
- Storage is FREE (smart move for stickiness)
- $2,500 credit for VC-funded startups

**Takeaway:** Zep charges for compute/processing, not storage. This encourages data accumulation (sticky).

---

### Mem0 (Direct Competitor - Memory Layer for LLMs)
| Plan | Price | Memories | API Calls/mo |
|------|-------|----------|--------------|
| Free | $0 | 10,000 | 1,000 retrieval |
| Starter | $19/mo | 50,000 | 5,000 retrieval |
| Pro | $249/mo | Unlimited | 50,000 retrieval |
| Enterprise | Custom | Unlimited | Unlimited |

**Pricing Model:** Tier-based with memory counts + API call limits
- Also offers open-source self-hosted option
- Claims 90% token reduction vs full-context methods
- Recently raised $24M (funded competitor)

**Takeaway:** Mem0's free tier (10,000 memories) is generous and creates investment. The jump to $249 for "unlimited" is steepâ€”opportunity for MemoryRouter.

### Pinecone (Vector Database - Adjacent Competitor)
| Plan | Price | Model |
|------|-------|-------|
| Starter | Free | 2GB storage, 2M writes, 1M reads/mo |
| Standard | $50/mo min | $0.33/GB storage + read/write units |
| Enterprise | $500/mo min | Custom, dedicated support |

**Pricing Model:** Pure usage-based (storage GB + operations)

- Read operations: Per read unit (RU)
- Write operations: Per write unit (WU)

**Takeaway:** Infrastructure-focused pricing. Works for technical users but can be confusing. MemoryRouter should be simpler.
